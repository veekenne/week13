Cat Hair Wars

Cat Hair Wars is a single issue web parody zine created by Vee Kennedy using AdobeFirefly and Canva in September of 2024 that takes homage from Siobhan o'Flynn, Alex Mitchell, and Mark Marino's For All Cat Kind, that Vee first learned about a ELO2024. The text featured in the zine pages are shortened versions of how Vee prompted Adobe Firely through the iterative process. You can learn more about the inspiring project here. Much like Fur All Cat Kind, Cat Hair Wars represents one creator's response to difficult newsmedia and the constant onslaught of 2024 with virtual cats as a coping mechanism.
