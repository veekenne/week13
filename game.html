Welcome to Cozy Clicky Kitties!
The base code for this game and the image were created in October 2024 using ChatGPT4o with canvas. More than fifty iterations between the LLM and the human facilitator, Vee Kennedy, were used to dictate the creation. Cozy Clicky Kitties works by calling p5.js and using clickable interaction fields. Try petting, feeding, and playing with each cat. Have fun!
